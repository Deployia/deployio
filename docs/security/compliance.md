# Compliance

Enterprise compliance standards and certifications for regulated industries.

## Compliance Standards

### SOC 2 Type II

- Security controls
- Availability monitoring
- Processing integrity
- Confidentiality measures

### ISO 27001

- Information security management
- Risk assessment procedures
- Incident response protocols
- Continuous improvement

### GDPR Compliance

- Data protection by design
- Right to be forgotten
- Data portability
- Consent management

### HIPAA (Healthcare)

- PHI protection
- Audit logging
- Encryption requirements
- Access controls

## Audit Logging

```bash
# View audit logs
deployio audit logs --since 30d

# Export compliance report
deployio compliance report --standard SOC2

# Monitor compliance status
deployio compliance status
```

## Data Governance

- Data classification
- Retention policies
- Backup procedures
- Geographic restrictions
